## 字符

| 字符 | 等同于           | 描述                         |
| ---- | ---------------- | ---------------------------- |
| .    | [^\n\r]          | 除了换行和回车之外的任意字符 |
| \d   | [0-9]            | 数字字符                     |
| \D   | [^0-9]           | 非数字字符                   |
| \s   | [ \t\n\x0B\f\r]  | 空白字符                     |
| \S   | [^ \t\n\x0B\f\r] | 非空白字符                   |
| \w   | [a-zA-Z_0-9]     | 单词字符(所有的字母)         |
| \W   | [^a-zA-Z_0-9]    | 非单词字符                   |



## 量词

| 代码  | 类型     | 描述                       |
| ----- | -------- | -------------------------- |
| ?     | 软性量词 | 出现零次或一次             |
| *     | 软性量词 | 出现零次或多次(任意次)     |
| +     | 软性量词 | 出现一次或多次（至道一次） |
| {n}   | 硬性量词 | 对应零次或者n次            |
| {n,m} | 软性量词 | 至少出现n次但不超过m次     |
| {n,}  | 软性量词 | 至少出现n次(+的升级版)     |



## 分组

中括号表示范围内选择，大括号表示重复次数。小括号允许我们重复多个字符。
